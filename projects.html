UMN Rocket Team

Rocket Avionics & Guidance, Navigation & Control

GNC & Avionics work on the UMN Rocket Team. Designed a custom backplane PCB in Altium using the CAN bus protocol and debugged with a logic analyzer. Built a Simulink flight state estimator using sensor fusion for flight stage detection — converted to C via Simulink codegen and deployed to the UFC flight computer. Separately designed a LoRa avionics bay PCB for telemetry downlink.
